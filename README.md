## 📱 What This App Does

A **Flutter mobile application** that encrypts and decrypts text messages using a secure key-based encryption system.

Built for real-world privacy use cases — users can encrypt sensitive text before sharing it, and decrypt it using the correct key.

| **Encryption** | Key-based text encryption |

| **Tools** | Android Studio, VS Code |

---

## ✅ Features

- 🔐 **Text Encryption** — encrypt any message with a secure key
- 🔓 **Text Decryption** — decrypt messages using the correct key
- 📋 **Copy to clipboard** — instantly copy encrypted/decrypted output
- 🎨 **Clean UI** — simple, intuitive interface
- ⚡ **Fast & lightweight** — optimized Flutter build
